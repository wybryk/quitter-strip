# quitter-strip
## Installation 
```npm install```
## Configuration
To run app you have to add firebase configuration file in store/.env.firebase.js
```
const FIREBASE_CONFIG = {
  apiKey: "",
  authDomain: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
  measurementId: ""
};

export default FIREBASE_CONFIG;
```
## Run
```ionic serve```
